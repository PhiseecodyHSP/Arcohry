# 代码规范

有条理, 风格统一的代码真的很重要啊!!!

为了规范代码编写, 在此有一些约定.


---
## 📛 命名规范

### 字段与函数
- 使用 **camelCase**: `storyButton`, `getName`
- 布尔值变量**不要**以 `is`, `has`, `can` 开头: `loading`, `extra`

### 局部变量
- 临时局部变量**可以**使用单字符或简短类名来命名:
  ```java
  for (int i = 0; i < list.size(); i++) {
      String str = list.get(i);
      doSomething(i, str);
      result.append(str).append('\n');
  }
- 局部变量较多时, 应当为重要的局部变量赋予有意义的名称:
  ```java
  int groupCount = groups.length;
  double average = 0.0D;
  for (int i = 0; i < groupCount; i++) {...}
  double variant = 0.0D;

### 静态常量
- 特别地, 使用 **UPPER_SNAKE_CASE**: `TEXT_COLOR`

### 类
- 使用 **PascalCase**: `StoryBranch`

### 文件与目录
- 包名使用 **camelCase**: `storyMode`
- 目录和文件名使用 **lower_snake_case**: `story_branches`, `hikari.png`

### 避免
- 无意义的缩写 (`usrCnt` → `userCount`)
- 用中文或拼音命名 (用一次我骂一次)

## 📐 格式约定

### 标点符号
- **一律用英文半角符号**, 并在该行后仍有文本时加**一个空格**, 顿号用逗号代替
- 括号外非标点文本与括号用**一个空格**分隔: `Arcaea (A Rhythm Game).`, `Rikka (Kawaii) is cute.`

### 空格与空行
- 中文与英文或数字之间用**一个空格**分隔
- 逻辑相关的代码块之间用**一个空行**分隔
- `//` 与注释内容之间用**一个空行**分隔
- 静态常量定义区域与成员字段定义区域之间用**一个空行**分隔
- 方法之间用**一个空行**分隔

### 换行缩进
- 左括号, 左方括号, 左尖括号**紧挨**左侧文本, 左花括号**隔一个空格**, 都**不换行**
- 语句中断缩进 **8 个空格**
- 方法第一个形参**紧挨**左括号, 若换行则**一行一个形参**, 并且**左对齐**

### 表达式和语句
- 双精度浮点数使用 `D` 后缀, 且总有**至少一位小数**: `1.0D`, `-3.4E-3D`
- 长整型使用 `L` 后缀: `2147483649L`
- 优先**针对结果类型**, 或以**隐式类型转换最少**的方式写数字: (`1 + doubleVar` → `1.0D + doubleVar`)
- 把字面量写在加法或乘法的**前面**: `1.0D + 2.0D * index`
- 引用**类成员字段或方法**统一加上 `this`: `this.enabled`, `this.run()`
- 过长的**传入参数**或**方法链式调用**考虑临时局部变量, 或者合理换行:
  ```java
  double result = calculateResult(user.getName(),
          user.getData().getExperience(),
          this.networkHandler.getInfo(),
          true, false);

## 📃 注释规范
### 基本
- 让我们说**中文**
- 遵循 [标点符号](#标点符号) 和 [空格与空行](#空格与空行) 的要求
- 一行文本长度控制在显示长度相当于 **80 个英文字符** 以下
- 换行**不应**截断一个短句, 应从**有停顿的地方** (如**标点符号**处) 换行

### Javadoc
- **详细说明**该字段 / 方法 / 类的**意义和作用**
- 对于方法, 应当**解释各个形参, 泛型形参, 返回值和可能抛出的异常**
- 对于类, 应当解释该类的**泛型形参, 详细用途和部分构造**, 标明**作者**

### 一般注释
- 通常来说, 以**行注释** `//` 的形式写在要注释的地方的**上一行**, 斜杠与内容**隔一个空格**
- 你也可以用**块注释** `/*...*/`
- **多行注释**既可以写**多个** `//`, 也可以使用**块注释**
