两个少女前往探索一个支离破碎的世界. 那里伴随着一个声音: 一段尘封的过往即将被揭开...

她们从这个除了废墟一无所有的世界中醒来, 却丝毫不记得这里曾发生过的一切.
这一刻她们才发现, 自己也不过是虚无的存在.
而接下来, 她们会发现一个名叫Arcaea的物体.
它由空中漂浮着玻璃质感般的碎片所积聚而成, 而这每一块碎片, 都包含着来自过去的鲜活回忆.
